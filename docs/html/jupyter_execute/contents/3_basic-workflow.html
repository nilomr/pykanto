Basic workflow

Basic workflow¶

from pykanto.utils.paths import pykanto_data, ProjDirs
from pykanto.dataset import SongDataset
from pykanto.parameters import Parameters

We are going to load one of the very small sample datasets that are packaged with pykanto—this will be enough for a first test to familiarise yourself with the package. See working with paths and directories to learn how to load your own data.

These are a few songs from two male great tits (Parus major) in my study population in Wytham Woods, Oxfordshire, UK.

DATASET_ID = "GREAT_TIT"
DIRS = pykanto_data(dataset=DATASET_ID)
print(DIRS)
